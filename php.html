Exercice 1 niveau 2

$viandes = array('poulet', 'boeuf', 'mouton','porc'); $cuissons = array('à point', 'bien cuit', 'saignant', 'bleu'); $commandes = array_combine($viandes, $cuissons);
print_r($commandes);

Qui nous donne : Array ( [poulet] => à point [boeuf] => bien cuit [mouton] => saignant [porc] => bleu )

Exercice 2 niveau 2

$genres="horreur fantastique action western thriller comédie drame romande historique";
function exploder($array) { echo "Le genre est:"; print_r (explode(" ",$array)); }
exploder($genres);

Qui nous donne : Le genre est:Array ( [0] => horreur [1] => fantastique [2] => action [3] => western [4] => thriller [5] => comédie [6] => drame [7] => romande [8] => historique )
